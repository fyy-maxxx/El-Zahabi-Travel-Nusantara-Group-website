## TROUBLESHOOTING DURING LAUNCH

| Issue | Solution |
|-------|----------|
| Site down | Check hosting status, restart server |
| Slow loading | Check server resources, optimize images |
| Form not sending | Check email config, test SMTP |
| CSS not loading | Clear cache, check file paths |
| JS errors | Check console, verify syntax |
| Image broken | Check file path, verify upload |
| Mobile broken | Check viewport meta tag, test responsive |
| 404 errors | Check routing, verify file exists |
| Security warning | Update SSL, check mixed content |
